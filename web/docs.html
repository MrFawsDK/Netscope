Python Modul

NetScope kan bruges direkte i Python scripts:

import netscope

# Ping eksempel
result = netscope.ping('google.com')
print(result)

# DNS opslag
dns = netscope.dns_lookup('google.com')
print(dns)

# Geolokation
geo = netscope.geoip('8.8.8.8')
print(geo)
  • ping(host): Tester svartid til en IP eller domæne
  • dns_lookup(domain): Finder IP-adresser for domæner
  • geoip(ip): Finder geografisk placering for IP

Komplet Guide

Trin-for-trin: Kom i gang med NetScope

  1. Installation: Følg installationsguiden for Python eller web.
  2. Opsætning: Start dashboardet eller importer Python-modulet.
  3. Ping Test: Indtast IP/domæne og tryk "Start Ping". Se statistik og historik.
  4. DNS Opslag: Indtast domæne og tryk "Slå op". Se IP-adresser og DNS info.
  5. Geolokation: Indtast IP og tryk "Find placering". Se kort og info.
  6. Avancerede Indstillinger: Skift tema, font, kontrast og gem præferencer.
  7. Historik & Eksport: Se tidligere tests og eksportér data.
  8. Support & Bidrag: Skriv til support eller bidrag på GitHub.

Eksempel på workflow

  • Installer NetScope med pip install netscope eller download web-dashboard.
  • Åbn dashboardet og vælg "Ping Test".
  • Indtast google.com og tryk start.
  • Se resultater, statistik og historik.
  • Skift til mørkt tema for bedre visning.
  • Eksportér data til CSV for rapportering.
